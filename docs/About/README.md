# 使用手册

## 仓库地址

源代码地址:[点击前往](https://github.com/xunless/CommonJS.git)

## 新建常用用法（##）

1. 打开 docs/Fun/README.md 文件夹
2. ##为方法名称，###为方法小分支
3. 左侧导航栏编译后自动生成，编写时只需关注 MD 文档格式

### 方法小分支（###）

###分支名字
